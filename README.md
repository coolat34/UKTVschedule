# UKTVschedule
This app provides program details for TV channels from the Freeview data file, containing approximately 266 channels. The user picks a favourite selection from the list of channels. The list is stored in a SwiftData file. The user can decide when the programs should start and if they should view programs for today or tomorrow. Those details are also stored. There is an additional option to click on a program title and view a pop-out containing further details. A timeline is also displayed.
